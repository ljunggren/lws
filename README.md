# Career Development — Wensheng

Grounded career planning for Wensheng, Core Architect of Boozang.

## What's here

- [`interview-guide.md`](./interview-guide.md) — **start here.** 90-minute structured interview to pick a career direction in the LLM-era software dev market. Six paths with upskilling specifics, a 1–5 rating baseline, and a decision matrix.
- [`career-advice.md`](./career-advice.md) — strategic memo (Mats's read): superpowers, growth edges, 12-month moves
- [`goals/`](./goals) — 3/12/36-month goals, seeded from the memo (to be rewritten after the interview picks a path)
- [`skills/`](./skills) — skills inventory grounded in the `bz` codebase
- [`learning-log/`](./learning-log) — running notes on courses, books, incidents, decisions

## How this was built

The memo in `career-advice.md` is built from two sources — not generic advice:

1. **`bz-business/wensheng.md`** — Mats's written profile of Wensheng's contributions
2. **`bz/.agent/memory/journal.md`** — concrete evidence of how Wensheng's code held up through Node 14→24, Mongoose 6→8, Puppeteer→Playwright migrations

Everything in `career-advice.md` points back to specific patterns, files, or incidents. No platitudes.
