# Wensheng — Career Development

A personal repo to track career goals, skills, and learning.

## Structure

- [`goals/`](./goals) — short- and long-term career goals
- [`skills/`](./skills) — skills inventory and growth areas

## How to use

Update entries regularly. Each folder has a template to get started.
